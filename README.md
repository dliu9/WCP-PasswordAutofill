# WCP Browser Settings — Passwords and Autofill Documentation

An interactive reference site documenting the **Microsoft Copilot (WCP) Browser Settings** for the **Passwords and Autofill** section, built for internal team review and design alignment.

## 🔗 Live Demo

**[View the interactive site →](https://dliu9.github.io/WCP-PasswordAutofill/WCPSetting_PasswordAutofill.html)**

Open in any browser — no login required.

---

## What's Inside

The site covers three sub-sections under **Settings → Passwords and Autofill**:

| Section | Description |
|---|---|
| **3A · Microsoft Password Manager** | Toggle settings, add/edit/delete passwords, import/export (CSV), password security check |
| **3B · Payment Methods** | Toggle settings, add/edit payment cards, CVV tooltip, billing address behaviour |
| **3C · Addresses and More** | Toggle settings, add/edit addresses, import (disabled) / export (JSON), bulk delete flow |

---

## How to Use

- Click any item in the **left sidebar** to navigate to that section
- Sections are colour-coded by feature area: **3A** (rust red) · **3B** (indigo) · **3C** (forest green)
- Each section shows exact field names, toggle states, required vs optional fields, and step-by-step flows
